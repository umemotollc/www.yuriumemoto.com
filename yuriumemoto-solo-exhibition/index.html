梅本佑利・個展「電話音楽」

2月18日~27日まで、西荻窪のMETEORにて、梅本佑利の個展、「電話音楽」を開催致します。

これまでの電話回線を用いた音楽作品群、「電話音楽」のアーカイブに加え、アーティスト李漢強（Lee Kan Kyo）とのコラボレーションによる冷蔵庫マグネット広告を題材にした新作（絵画作品＋電話音楽）３作品を展示致します。

参加演奏家：阪田知樹（ピアニスト）、成田達輝（ヴァイオリニスト）、丁仁愛（フルーティスト）

新作3作品は、水道修理や害獣駆除などの冷蔵庫用広告マグネットを題材に（梅本による音楽・スケッチを元に）李漢強が書き下ろし。各作品には実際に電話を掛けることができる番号が記載されています。

また、これら絵画作品を元に、実際のマグネットによるアート作品も制作。会期中、リミテッドで販売予定です。

梅本佑利 (うめもとゆうり),作曲家  / Yuri Umemoto

 

プロフィール：

作曲家。2002年、東京生まれ、神田育ち。アキバ系カルチャーに影響を受けて幼少、思春期を過ごす。西洋音楽、現代音楽のジャンルで、日本の消費社会、ゲーム、アニメなどのサブカルチャーを扱った作曲活動を行い、作品は国内外の著名な演奏家によって、数多く演奏されている。ヤマハ株式会社との共同開発によるAI合奏システム・自動演奏技術を用いた新作を発表するなど、コラボレーションにも積極的である。東京音楽大学付属高等学校を経て、現在、愛知県立芸術大学（作曲）に在学。

https://www.yuriumemoto.com/

李漢強 (リーカンキョウ), グラフィック・アーティスト / Lee Kan Kyo

 

プロフィール：

アーティスト。台湾出身。東京造形大学大学院（造形専攻）修了。幼い頃から見ていたTV番組で、日本のエンターテインメントに魅力を感じ、2007年来日。スーパーのチラシ、週刊誌、ポイントカードなど消費社会の現象に着目した創作で、第１０回グラフィック「1_WALL」グランプリ受賞。

https://www.leekankyo.com/

阪田知樹（さかたともき）, ピアノ / Tomoki Sakata

 

プロフィール：

2021年エリザベート王妃国際音楽コンクール第4位入賞。2016年フランツ・リスト国際ピアノコンクール第1位、6つの特別賞。

東京芸術大学を経て、ハノーファー音楽演劇大学大学院ソリスト課程に在籍。

第14回ヴァン・クライバーン国際ピアノコンクールにて弱冠19歳で最年少入賞。

ピティナ・ピアノコンペティション特級グランプリ、聴衆賞等5つの特別賞、クリーヴランド国際ピアノコンクールにてモーツァルト演奏における特別賞、キッシンジャー国際ピアノオリンピックでは日本人初となる第1位及び聴衆賞。

国内はもとより、世界各地20カ国で演奏を重ね、国際音楽祭への出演多数。

2015年CDデビュー、2020年3月、世界初録音を含む意欲的な編曲作品アルバムをリリース。内外でのテレビ・ラジオ等メディア出演も多い。

2017年横浜文化賞文化・芸術奨励賞受賞。

https://www.japanarts.co.jp/artist/tomokisakata/

成田達輝（なりたたつき）, ヴァイオリン / Tatsuki Narita

 

プロフィール：

1992年生まれ。札幌で3歳よりヴァイオリンを始める。 ロン＝ティボー国際コンクール(2010)エリザベート王妃国際音楽コンクール(2012)、仙台国際音楽コンクール(2013)でそれぞれ第2位受賞。 これまでに、ペトル・アルトリヒテル、オーギュスタン・デュメイ、ピエタリ・インキネンなど著名指揮者や国内外オーケストラと多数共演している。 2018年8月と翌2月に韓国で行われた平昌音楽祭に参加し、ソン・ヨルム、スヴェトリン・ルセヴらと共演。2018年にはミンスクで行われたユーリ・バシュメット音楽祭にも参加している。 使用楽器は、アントニオ・ストラディヴァリ黄金期の”Tartini” 1711年製。(宗次コレクションより貸与)。

https://www.japanarts.co.jp/artist/tatsukinarita/

丁仁愛（チョンインエー）, フルート / Inae Jeong

 

プロフィール：

東京生まれ。東京藝術大学器楽科卒業。フルートを前川美穂、立川和男、堀井恵、高木綾子、斎藤和志、小池郁江の各氏に、フラウト・トラヴェルソを前田りり子氏に師事。2017年 小澤征爾音楽塾オペラ・プロジェクトXV参加。2018年 木ノ脇道元プロデュース演奏会「若手プレーヤー達と。」参加。2019年 東京藝術大学主催”Tokyo Independent ”にて絵画出展。第16回 21世紀音楽の会出演など。現在は都内を中心に演奏と作曲、編曲、詩作、描画での創作活動を行なう。現代奏造Tokyoメンバー。

https://inaejeong.amebaownd.com
